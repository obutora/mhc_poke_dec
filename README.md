# poke_dec

ポケモンでリスト操作を学ぼう！
ついでにjson操作も学ぼう！

## description
そこに ３びき
ポケモンが いる　じゃろう！

すきなのを　おまえに　やろう！

ということで、
２つのjsonファイルを組み合わせて１つのリストを生成し、
ランダムに３つのポケモンを選ぶアプリを作成しました。
